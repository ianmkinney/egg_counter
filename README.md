A retro-style counter simulation that displays receipts coming in from the left and animates a classic flap counter on the right as eggs are counted from menu items.

## How to Use

1. Open `index.html` in a web browser
2. Click **START** to begin the simulation
3. Watch as receipts appear on the left and the counter ticks up on the right
4. Click **STOP** to pause the simulation
5. Click **RESET** to clear all receipts and reset the counter to zero

## Files

- `index.html` - Main HTML structure
- `styles.css` - Retro styling and animations
- `script.js` - Counter logic and receipt generation
- `PHYSICAL-VERSION.md` - Ideas and plans for building a physical version

## Menu Items

The simulation includes 30+ breakfast menu items with realistic egg counts:
- Scrambled Eggs (2 eggs)
- Omelets (3 eggs)
- Eggs Benedict (2 eggs)
- Breakfast Burritos (2 eggs)
- And many more!

## Customization

You can customize the simulation by editing `script.js`:
- Add more menu items in the `menuItems` object
- Adjust receipt generation frequency (currently 2-5 seconds)
- Modify customer names
- Change the number of digits in the counter

## Browser Compatibility

Works best in modern browsers (Chrome, Firefox, Edge, Safari) with CSS3 and JavaScript ES6+ support.
